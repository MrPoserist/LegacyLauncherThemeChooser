# LegacyLauncherThemeChooser
Easy to use theme chooser for Legacy Launcher.

![screenshot.png](https://github.com/MrPoserist/LegacyLauncherThemeChooser/blob/280c3c34a09bdb1b3ebf5e14ed7d8cdb7dcecb63/screenshot.png)

# Run from source
You need to install **python3**, **tkinter** and **sv-ttk**.
Then, run **ui.py** file.
